- Part a
	- An example with n = 2
		- $$x = \begin{bmatrix} x_1 \\ x_2 \end{bmatrix}, \quad \mathbf{A} = \begin{bmatrix} A_{1,1} & A_{1,2} \\ A_{2,1} & A_{2,2} \end{bmatrix}$$
		- Step 1 : compute $\mathbf{A}x$ f
			- $$\mathbf{A}x = \begin{bmatrix} A_{1,1} x_1 + A_{1,2} x_2 \\ A_{2,1} x_1 + A_{2,2} x_2 \end{bmatrix}$$
		- Step 2Compute $x^\top (\mathbf{A}x)$ , observe $x^\top$ is a row vector and $\mathbf{A}x$ is a column vector therefore the result is going to be a scalar.
		  $$x^\top \mathbf{A}x = x_1(A_{1,1} x_1 + A_{1,2} x_2) + x_2(A_{2,1} x_1 + A_{2,2} x_2)$$
		  $$= x_1 A_{1,1} x_1 + x_1 A_{1,2} x_2 + x_2 A_{2,1} x_1 + x_2 A_{2,2} x_2$$
		- based on this pattern is $x_i \cdot A_{i,j} \cdot x_j$ for some $i, j$.
		  $$x^\top \mathbf{A} x = \sum_{i=1}^{n} \sum_{j=1}^{n} x_i \, A_{i,j} \, x_j$$
		- Step 3: Similarly it can be proven that $y^\top \mathbf{B} x$ — every term is $y_i \cdot B_{i,j} \cdot x_j$:
		  $$y^\top \mathbf{B} x = \sum_{i=1}^{n} \sum_{j=1}^{n} y_i \, B_{i,j} \, x_j$$
		- Final result
		  $$f(x,y) = \sum_{i=1}^{n} \sum_{j=1}^{n} x_i \, A_{i,j} \, x_j + \sum_{i=1}^{n} \sum_{j=1}^{n} y_i \, B_{i,j} \, x_j + c$$
- Part b
	- To find the $k$-th component of the gradient with respect to $x$, take $\frac{\partial f}{\partial x_k}$:
	- using example with n = 2 from part a.
	  $$x^\top \mathbf{A}x = x_1 A_{1,1} x_1 + x_1 A_{1,2} x_2 + x_2 A_{2,1} x_1 + x_2 A_{2,2} x_2$$
	- Consider $\frac{\partial f}{\partial x_1}$: for example calculation (still with n = 2 sample from a)
		- $x_1 A_{1,1} x_1$ => $A_{1,1} x_1^2$ derivative = $2A_{1,1} x_1$
		- $x_1 A_{1,2} x_2$ => derivative = $A_{1,2} x_2$ ($x_2$ is constant in terms of $x_1$)
		- $x_2 A_{2,1} x_2$ => derivative = $x_2 A_{2,1}$ ($x_2$ is constant in terms of $x_1$)
		- $x_2 A_{2,2} x_2$ => derivative = 0 ($x_2$ is constant in terms of $x_1$)
	- Therefore
	  $$ \frac{\partial f}{\partial x_1} (x^\top \mathbf{A} x) =  2A_{1,1} x_1 + A_{1,2} x_2 + x_2 A_{2,1}$$
		- rewriting for two series because $x_k$ contributes to the derivative twice, once as left term and once as a right term. For n= 2 example
		  $$ \frac{\partial f}{\partial x_1} (x^\top \mathbf{A} x) =  (A_{1,1} x_1 + A_{1,2} x_2) +  (A_{1,1} x_1 + A_{2, 1} x_2) $$
	- for k
		- When $i = k$: contributes $\sum_j A_{k,j} x_j$ ($x_k$ appears as the left factor)
		- When $j = k$: contributes $\sum_i x_i A_{i,k}$ ($x_k$ appears as the right factor)
		- $\frac{\partial}{\partial x_k}(x^\top \mathbf{A} x) = \sum_j A_{k,j} x_j + \sum_i A_{i,k} x_i$
	- for the term $y^\top \mathbf{B} x$
		- $\sum_i \sum_j y_i B_{i,j} x_j$ — only involves $x_j$, so differentiate w.r.t. $x_k$ ($j = k$)
		- $\frac{\partial}{\partial x_k}(y^\top \mathbf{B} x) = \sum_i y_i B_{i,k}$
	- c gets a zero derivative
	  $$\left[\nabla_x f\right]_k = \sum_{j=1}^n A_{k,j}\, x_j + \sum_{i=1}^n A_{i,k}\, x_i + \sum_{i=1}^n y_i\, B_{i,k}$$
	- For $\nabla_y f$: take $\frac{\partial f}{\partial y_k}$
		- $x^\top \mathbf{A} x$ has no $y$ terms $\Rightarrow$ derivative is 0
		- $y^\top \mathbf{B} x$: differentiate $\sum_i \sum_j y_i B_{i,j} x_j$ w.r.t. $y_k$ ($i = k$): $\frac{\partial}{\partial y_k} = \sum_j B_{k,j} x_j$
		- c gets a zero derivative
		- $$\left[\nabla_y f\right]_k = \sum_{j=1}^n B_{k,j}\, x_j$$
	- Final Result
	  $$\left[\nabla_x f\right]_k = \sum_{j=1}^n A_{k,j}\, x_j + \sum_{i=1}^n A_{i,k}\, x_i + \sum_{i=1}^n y_i\, B_{i,k}$$
	  $$\left[\nabla_y f\right]_k = \sum_{j=1}^n B_{k,j}\, x_j$$
- Part C
	- from part b we have
	  $$[\nabla_x f]_k = \sum_j A_{k,j} x_j + \sum_i A_{i,k} x_i + \sum_i y_i B_{i,k}$$
	- Step 1 : Identify equivalent matrix operation for  $\sum_j A_{k,j} x_j$
		- Its equivalent to taking row $k$ of $\mathbf{A}$, dot with $x$ — that's the $k$-th entry of $\mathbf{A}x$.
		- Example ($n=2$, $k=1$): $A_{1,1} x_1 + A_{1,2} x_2$ = row 1 of $\mathbf{A}$ dotted with $x$ = $(\mathbf{A}x)_1$
	- Step 2 : Identify equivalent matrix operation for  $\sum_i A_{i,k} x_i$
		- Its equivalent to taking <strong>column</strong> $k$ of $\mathbf{A}$, dot with $x$. Column $k$ of $\mathbf{A}$ = row $k$ of $\mathbf{A}^\top$.
		- Example ($n=2$, $k=1$): $A_{1,1} x_1 + A_{2,1} x_2$ = column 1 of $\mathbf{A}$ dotted with $x$ = $(\mathbf{A}^\top x)_1$
	- Step 3 :Identify equivalent matrix operation for  $\sum_i y_i B_{i,k}$
		- Same pattern as step 2 but with $y$ and $\mathbf{B}$: column $k$ of $\mathbf{B}$ dotted with $y$.
		- row $k$ of $\mathbf{B}^\top$ dotted with $y$ — the $k$-th entry of $\mathbf{B}^\top y$.
		- Adding all the parts
		  $\nabla_x f$ for each component k is $(\mathbf{A}x)_k + (\mathbf{A}^\top x)_k + (\mathbf{B}^\top y)_k$
		  $$\nabla_x f(x,y) = \mathbf{A}x + \mathbf{A}^\top x + \mathbf{B}^\top y = (\mathbf{A} + \mathbf{A}^\top)x + \mathbf{B}^\top y$$
	- Step 4: With the same analysis for $\nabla_y f$  
	  $\sum_j B_{k,j} x_j$ is the $k$-th entry of $\mathbf{B}x$
		- $$\nabla_y f(x,y) = \mathbf{B}x$$
	- Final result
		- $$\nabla_x f(x,y) = \mathbf{A}x + \mathbf{A}^\top x + \mathbf{B}^\top y = (\mathbf{A} + \mathbf{A}^\top)x + \mathbf{B}^\top y$$
		- $$\nabla_y f(x,y) = \mathbf{B}x$$
