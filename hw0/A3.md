- Part a
	- Known
		- $X_1, X_2, \ldots, X_n \sim \mathcal{N}(\mu, \sigma^2)$ i.i.d.
		- $\mathcal{N}(0, 1) => \mu = 0, \sigma^2 = 1$
		- $X1 \sim \mathcal{N}(\mu, \sigma^2)$
	- Step 1 : Map mean and variance to new equations
		- if $X_n \sim \mathcal{N}(\mu, \sigma^2)$, then $aX_n + b \sim \mathcal{N}(a\mu + b, a^2\sigma^2)$
		- Mean : $a\mu + b = 0$
		- Variance : $a^2\sigma^2 = 1$
			- $a^2 = \frac{1}{\sigma^2}$, so $a = \frac{1}{\sigma}$
			- Replacing a's value in Mean equation
			- $\frac{\mu}{\sigma} + b = 0$, so $b = -\frac{\mu}{\sigma}$
	- Final result
		- $a = \frac{1}{\sigma}$
		- $b = -\frac{\mu}{\sigma}$
- Part b.

		- $a = \frac{1}{\sigma}$
		- $b = -\frac{\mu}{\sigma}$
		- standard PDF  : $\varphi(z) = \frac{1}{\sqrt{2\pi}} e^{-z^2/2}$
			- $\varphi(z)$ is symmetric around 0 (symmetric bell curve at 0)
	- Step 1 : Simplify what's inside the absolute value
		- from (a) we know aX1 + b = $\sim  \mathcal{N}(0, 1)$, is standard normal random variable. call it Z
	- Step 2 :  Expected value of a function of a continous random variable is integrating the function times the PDF.
	  $$\mathbb{E}[|Z|] = \int_{-\infty}^{\infty} |z| \cdot \varphi(z)\, dz$$
		- split the integral and use the symmetry to simplify.
		  $$\mathbb{E}[|Z|] = \int_{-\infty}^{0} (-z) \varphi(z)\, dz + \int_{0}^{\infty} z\, \varphi(z)\, dz$$
		  $$\mathbb{E}[|Z|] = 2 \int_{0}^{\infty} z\, \varphi(z)\, dz$$
		  $$= 2 \int_{0}^{\infty} z \cdot \frac{1}{\sqrt{2\pi}} e^{-z^2/2}\, dz$$
		-
	- Step 3 : Evaluate using u-substitution
		- if we take u = $z^2/2$ , then du = $z dz$
		- rewrite
		  $$\mathbb{E}[|Z|] = \frac{2}{\sqrt{2\pi}} \int_{0}^{\infty} e^{-u}\, du$$
		- The antiderivative of $e^{−u}$ is $−e^{-u}$ is
		- When $z=0$, $u=0$; when $z \to \infty$, $u \to \infty$
		  $$= \frac{2}{\sqrt{2\pi}} \Big[-e^{-u}\Big]_0^{\infty}$$
		  $$\mathbb{E}[|Z|] = \frac{2}{\sqrt{2\pi}} \Big[-e^{-u}\Big]_0^{\infty}$$
		  $$\mathbb{E}[|Z|] = \frac{2}{\sqrt{2\pi}} \cdot 1 = \frac{2}{\sqrt{2\pi}}$$
		- Final Result

- Part c
	- Known:
	- $X_1, X_2$ are i.i.d. $\sim \mathcal{N}(\mu, \sigma^2)$
	- Additivity of Variance : for two independent random variables X and Y, the variance of their sum is the sum of their individual variances. $\text{Var}[X1 + X2] = \text{Var}[X_1] + \text{Var}[X_2] = \sigma^2 + \sigma^2 = 2\sigma^2$
	- and $\text{Var}[aX] = a^2\sigma^2\\$
	- Using linearity of expectation: the expectation of a sum of X & Y random variables is the sum of their individual expectations
	  $$\mathbb{E}[X_1 + 2X_2] = \mathbb{E}[X_1] + 2\,\mathbb{E}[X_2] = \mu + 2\mu = 3\mu$$
	- Additivity of Variance
	  $$\text{Var}[X_1 + 2X_2] = \text{Var}[X_1] + \text{Var}[2X_2] = \sigma^2 + 4\sigma^2 = 5\sigma^2$$
	- Final answer
	  $$\mathbb{E}[X_1 + 2X_2] = 3\mu$$
	  $$\text{Var}[X_1 + 2X_2] = 5\sigma^2$$
- Part d
	- Known:  $\hat{\mu}_n = \frac{1}{n} \sum_{i=1}^{n} X_i$
	- Step 1: Expected Value : $\mathbb{E}[\hat{\mu}_n]$ 
	  $$
	  \mathbb{E}[\hat{\mu}_n] = \mathbb{E}\!\left[\frac{1}{n} \sum_{i=1}^n X_i\right] \\
	  \\
	  \mathbb{E}[\hat{\mu}_n] = \frac{1}{n} \mathbb{E}\!\left[\sum_{i=1}^n X_i\right] \\
	  Sum of n samples = n * mean\\
	  \mathbb{E}[\hat{\mu}_n] = \frac{1}{n} \cdot n\mu = \mu
	  $$
	- Step 2 : Variance : $\text{Var}[\hat{\mu}_n]$
		- Additivity of Variance & $\text{Var}[aX]$ rules above
		  
		  $$
		  \text{Var}[\hat{\mu}_n] = \text{Var}\!\left[\frac{1}{n} \sum_{i=1}^n X_i\right] \\
		  \text{Var}[\hat{\mu}_n] =\frac{1}{n^2} \text{Var}\!\left[\sum_{i=1}^n X_i\right] \\
		  \text{Var}[\hat{\mu}_n] =  \frac{1}{n^2} ( n\sigma^2 ) = \frac{\sigma^2}{n}
		  $$
- Step 3 :Compute $\mathbb{E}[\sqrt{n}(\hat{\mu}_n - \mu)]$
  $$= \sqrt{n} \cdot (\mathbb{E}[\hat{\mu}_n] - \mu) = \sqrt{n} \cdot (\mu - \mu) = 0$$
- Step 4: Compute $\text{Var}[\sqrt{n}(\hat{\mu}_n - \mu)]$
  $$= (\sqrt{n})^2 \cdot \text{Var}[\hat{\mu}_n - \mu] = n \cdot \text{Var}[\hat{\mu}_n] = n \cdot \frac{\sigma^2}{n} = \sigma^2$$
- Final Result
  $$\mathbb{E}\!\left[\sqrt{n}(\hat{\mu}_n - \mu)\right] = 0, \quad \text{Var}\!\left[\sqrt{n}(\hat{\mu}_n - \mu)\right] = \sigma^2$$
