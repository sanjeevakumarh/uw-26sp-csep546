- Part a
	- Known
		- vectors v and w in $R^n$, with all $v_i \neq 0$
		- function g where $g (v_i)  = w_i$
		- To invert any matrix M, we need a matrix $M^{-1}$ such that $M⋅M^{−1}=I$
		- For a diagonal matrix — flip each diagonal entry by $\frac{1}{x_{i,i}}$
	- let diag(v) be  - representing only n=3
	  $$\mathbf{V} = \begin{bmatrix} v_{11} & 0 & 0 \\ 0 & v_{22} & 0 \\ 0 & 0 & v_{33} \end{bmatrix}$$
	- Now $V^{-1} = diag(v)^{-1}$will be
	  $$\begin{bmatrix} 1/v_{11} & 0 & 0 \\ 0 & 1/v_{22} & 0 \\ 0 & 0 & 1/v_{33} \end{bmatrix}$$
	- For $diag(v)^{-1} = diag(w)$ to be true we need $v_i = 1/ w_i$ for all $i$
	- Therefore $g(v_i) = 1/v_i$
	- Final result
		- $g(x) = 1/x$
- Part b
	- **Known:** $\mathbf{A}$ is orthonormal: $\mathbf{A}\mathbf{A}^\top = \mathbf{A}^\top\mathbf{A} = \mathbf{I}$
	- **Proof:**
	- Step 1 - Squared L2  norm
	   $\|\mathbf{A}x\|_2^2 = (\mathbf{A}x)^\top(\mathbf{A}x)$
	- Step 2 - Expand $(Ax)^\top = x^\top A^\top$
	  $\|\mathbf{A}x\|_2^2  = x^\top \mathbf{A}^\top \mathbf{A} x$ 
	  $(\mathbf{A}x)^\top = x^\top \mathbf{A}^\top$
	- Step 3 - since $\mathbf{A}^\top \mathbf{A} = \mathbf{I}$
	  $\|\mathbf{A}x\|_2^2  =  x^\top \mathbf{I} x$ 
	                   $= x^\top x = \|x\|_2^2 $
	- **Final Result**
	  $$\|\mathbf{A}x\|_2^2 = \|x\|_2^2$$

- Part c
	- **Known:** $\mathbf{B}$ is invertible and symmetric ($\mathbf{B} = \mathbf{B}^\top$)
	- **Proof:**
	                - Step 1: $\mathbf{B} \cdot \mathbf{B}^{-1} = \mathbf{I}$
	  
	                - Take transpose of both sides:
	  
	                - $(\mathbf{B} \cdot \mathbf{B}^{-1})^\top = \mathbf{I}^\top$
	  
	                - $(\mathbf{B}^{-1})^\top \cdot \mathbf{B}^\top = \mathbf{I}$
	- Replace $\mathbf{B}^\top = \mathbf{B}$
	  $(\mathbf{B}^{-1})^\top \cdot \mathbf{B} = \mathbf{I}$
	- meaning $(\mathbf{B}^{-1})^\top$ is a left inverse of $\mathbf{B}$. Since $\mathbf{B}$ is invertible, the left inverse equals the inverse:
	  
	                - $(\mathbf{B}^{-1})^\top = \mathbf{B}^{-1} \quad$
	- Therefore
	                - $\mathbf{B}^{-1}$ is symmetric.
